## Identificación de la Tecnología de Almacenamiento

El sistema cuenta con dos dispositivos de almacenamiento: un HDD y un SSD NVMe. Las pruebas se realizaron sobre el HDD, por lo que los resultados corresponden a esa tecnología.”

| Sistema operativo      | Windows 11              |
